The source code is available at https://github.com/pawzy/MoodleTest

* The folder Gnuplot contains the script used to graph atop logs.
* The folder Testresults contains graphs created, atop monitoring results and the results from moodle metrics. 
* The folder Traces contains some of the traces of test runs. 

Current version of this project is running on .NET 4.6.1 and F# runtime 4.5.0.0. 
The current version of NModel is not yet uploaded to github but can be shared through .dll file (https://github.com/juhan/NModel - previous version of NModel).
